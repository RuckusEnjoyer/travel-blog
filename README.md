## Description

üîç A Travel Blog made using node.js, express.js, mySQL, Sequelize ORM and handlebars.js that allows users add places they have been and to write about their experiences  
  
![Travel Blog]()

## User Story

```
As person who likes to travel, I want to track my places traveled, add what I did in these places, and see other peoples experience in that destination and in others.
```

üíæ
